## 2장. 특징,함수 및 객체
### 언어의 특징
####참조와 값
+ 변수는 값을 복사하거나 참조하는 방식으로 데이터를 담는다.
+ 기본값(문자열,숫자,부울,빈 값,정의되지 않은 값) 외 에는 변수에 객체를 참조한다.
+ 객체는 속성(property)와 방법(method)를 갖는다.
```javascript
// 배열 비교
var arr = ['1','2','3'];
var arr2 = arr;

arr = ['a','b'];

// arr이 새로운 객체를 참조하여 false
console.log( arr !== arr2);

// 문자열 객체 비교
var str = 'str';
var str2 = str;

// 새로운 객체가 만들어짐
str += 'ing';

// 두 객체는 다르다.
console.log( str !== str2);
```
